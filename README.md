# Weatherly

A very basic weather app built with plain HTML, CSS, and JavaScript.

- Search any city
- Use browser geolocation
- Powered by free Open-Meteo API (no key required)

## Run locally

Just open `index.html` in a browser.

## Deployed

Deployed automatically to Vercel via GitHub.

## Tech

- Tailwind CSS (CDN)
- Font Awesome icons
- Open-Meteo geocoding + forecast APIs

Created and deployed using GitHub + Vercel MCP tools.
